# Ramen & Roll — Landing Page
 
A responsive landing page for a fictional Japanese restaurant called **Ramen & Roll**. This project was built as part of my frontend learning journey using HTML, CSS and a small amount of JavaScript.
 
The goal was to practice layout, responsive design, accessibility basics and simple UI interactions.
 
---
 
## Preview
 
The website presents a modern and minimal interface inspired by Japanese food culture. Based on a Pinterest picture I saw.
 
Main sections include:
 
- Navigation header with mobile hamburger menu
- Hero section introducing the brand
- Scrollable bowls gallery (horizontal scrolling was a thing I wanted to test here)
- Responsive footer with company information
 
---
 
## Technologies
 
- **HTML5** — semantic structure
- **CSS3** — layout, responsive design and UI styling
- **JavaScript (Vanilla)** — small interactions and animations
 
Key concepts practiced:
 
- Semantic HTML
- CSS Custom Properties (variables)
- Responsive design with `clamp()` and media queries
- Flexbox and Grid
- Scroll animations
- Mobile navigation menu (hamburger)
- Basic accessibility attributes (aria)
 
---
 
## Features
 
- Responsive layout for desktop, tablet and mobile
- Sticky navigation header
- Animated navigation underline
- Mobile hamburger menu
- Smooth horizontal scroll gallery
- Clean modern UI design
- Fluid typography and spacing
